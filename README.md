# 🚂 Shunting Yard Algorithm - Visualizador Interactivo

¡Bienvenido! Este es un proyecto interactivo construido en Python utilizando el framework **Flet**. Su objetivo es visualizar paso a paso cómo funciona el famoso **Algoritmo Shunting Yard** (creado por Edsger Dijkstra) para analizar expresiones matemáticas.

🔗 **[¡Prueba la aplicación en vivo aquí!](https://jxsewv.github.io/shunting-yard-flet/)**

## 🌟 Características
* **Visualización Paso a Paso:** Observa cómo los números y operadores entran y salen de la Pila (Stack) y la Cola de Salida.
* **Interfaz Gráfica Atractiva:** Construida enteramente con Python y Flet, sin necesidad de escribir HTML o CSS.
* **Responsivo:** Funciona perfectamente tanto en navegadores de escritorio como en dispositivos móviles.
